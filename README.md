# Smartill_project
Le projet consiste à intégrer et automatiser la gestion d’un tiroir-caisse intelligent SmartTill dans un système de point de vente (POS). L’objectif principal est de suivre en temps réel les mouvements d’argent liquide (ventes, remboursements, dépôts, retraits) tout en éliminant les comptages manuels.
